##### Under / Over kill for keyword searching on web page
1. Ignore tags - Tags we decide to ignore, in order to only search the keyword on page's visible part
2. Under kill - keyword on the page's ignore tag list, also on the visible part
  - In this scenario, even when keyword is on the visible part of the page, because it's in the ignore tag, it will not be captured
  - https://www.imdb.com/
3. Over kill - keyword not on the page's ignore tag list, and on the invisible part
  - In this scenario, keyword is on the invisible part of the page, and still, it will be captured as keyword
  - https://www.imdb.com/
